# Dossier Public - Assets Statiques

Ce dossier contient les fichiers statiques qui seront servis à la racine de votre application.

## Logo

Placez votre logo ici :
- **Fichier PDF** : `NL Solutions - Logo Main.pdf`
- **Fichier Image (recommandé)** : `NL Solutions - Logo Main.png` ou `NL Solutions - Logo Main.svg`

### Conversion PDF vers Image

Pour convertir le PDF en image, vous pouvez utiliser :
- Outils en ligne : https://www.ilovepdf.com/pdf-to-png
- Adobe Acrobat
- Outils de ligne de commande comme ImageMagick

**Recommandation** : Utilisez une version PNG ou SVG pour une meilleure compatibilité avec les navigateurs web.
