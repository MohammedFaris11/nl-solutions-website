# Image pour le Secteur Plasturgie

Pour ajouter une image pour le projet "Secteur Plasturgie" :

1. Trouvez une image appropriée représentant l'industrie de la plasturgie (machines d'injection, production de plastique, etc.)
2. Nommez le fichier : `plasturgie.jpg` (ou `plasturgie.png`)
3. Placez-le dans le dossier `public/`
4. Formats recommandés : JPG, PNG
5. Dimensions recommandées : 1200x800px ou plus

## Sources d'images gratuites :
- Unsplash : https://unsplash.com/s/photos/plastic-injection-molding
- Pexels : https://www.pexels.com/search/plastic%20manufacturing/
- Pixabay : https://pixabay.com/images/search/plasturgie/

## Image actuelle
L'image utilise actuellement une URL Unsplash en ligne. Pour utiliser une image locale, assurez-vous que le fichier s'appelle `plasturgie.jpg` et se trouve dans le dossier `public/`.
